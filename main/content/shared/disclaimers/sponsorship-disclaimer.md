# Sponsorship Disclaimer

status: draft

## Purpose

This disclaimer is intended for use on bisulfid.com pages that carry sponsored reference placement or supplier visibility content (Phase 03 revenue layer). When published, it will clearly distinguish between editorial reference content and sponsored content, ensuring that sponsored material does not compromise the reference integrity or trust authority of the page.

Per `doctrine/PROJECT_DOCTRINE.md`, sponsored content is prohibited on `/acquire/`, glossary pages, and authority reference pages. This disclaimer applies only to approved public gateway pages where limited sponsorship is permitted.

## Warning

This file is a draft. It is not yet attached to any published page or route.

Do not include this disclaimer in any public-facing content until it has been reviewed, approved, and linked to a published route in `main/data/routes.json`.

Sponsored content that is not clearly distinguished from editorial content is prohibited.
