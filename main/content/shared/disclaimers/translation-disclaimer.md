# Translation Disclaimer

status: draft

## Purpose

This disclaimer is intended for use on bisulfid.com multilingual pages (DE, AR, ZH, JA, FR, ES). When published, it will communicate that translations on bisulfid.com are governed reference layers, not machine-translated content, and that the English version is the source layer for all factual claims. It will state that translated pages have been reviewed against the claim registry and that any discrepancy between language versions should be resolved by reference to the English source.

This disclaimer is required on all non-English language pages before they may pass multilingual integrity (Quality Gate 11).

## Warning

This file is a draft. It is not yet attached to any published page or route.

Do not include this disclaimer in any public-facing content until it has been reviewed, approved, and linked to a published route in `main/data/routes.json`.

No language layer publishes until this disclaimer is reviewed and attached.
