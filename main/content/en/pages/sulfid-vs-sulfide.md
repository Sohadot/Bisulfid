# Sulfid vs Sulfide: The Broader Pattern

The bisulfid/bisulfide spelling boundary is not unique. It reflects a broader systematic difference between German and English chemical nomenclature for compounds in the -id/-ide suffix class.

---

## The Pattern

German chemical naming conventions for certain compound classes use -id where English uses -ide.

Examples where this pattern appears [SOURCE REQUIRED]:

- Sulfide (EN) / Sulfid (DE)
- Oxide (EN) / Oxid (DE)
- Chloride (EN) / Chlorid (DE)

These examples are included as illustrative of the claimed pattern. They are marked [SOURCE REQUIRED] because formal nomenclature authority citations are needed before any of these can be stated as verified fact on a public-facing page.

## Why This Matters for bisulfid.com

The German identity layer of this asset is not arbitrary. The domain name itself — bisulfid — derives from this naming pattern. Understanding that the -id suffix is the German form, not an error, is prerequisite to understanding what bisulfid.com is.
