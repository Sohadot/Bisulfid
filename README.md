## The Terminology System

bisulfid.com does not own one chemical term.  
It controls the **center of a sulfur terminology system.**

```
                    ┌─────────────┐
                    │   Bisulfid  │  ← center node · German-form · HS⁻
                    │   bisulfid.com
                    └──────┬──────┘
           ┌───────────────┼───────────────┐
           ↓               ↓               ↓
       Bisulfide       Hydrosulfide     Sulfid
      (EN boundary)    (IUPAC form)   (DE root)
           ↓               ↓               ↓
     Sodium Bisulfide   Hydrogen      Disulfide
         NaHS           Sulfide H₂S    −S−S−
           ↓                               ↓
    Industrial Systems           Proteins · Rubber · MoS₂
```

Every term is a governed entity in `ontology/sulfur_terms.json`  
with `language_vector`, `chemical_field`, `risk_level`, `spatial_node`, and `verified_sources`.

## The Source Discipline

Every factual claim requires a verified entry in `main/data/sources/source_registry.json`.

**Pre-approved sources:**

| ID | Source | Authority |
|---|---|---|
| SRC-001 | Merriam-Webster | Terminology · First known use dates |
| SRC-002 | Oxford English Dictionary | Historical etymology |
| SRC-005 | IUPAC Red Book | Chemical naming standards |
| SRC-006 | Duden | German-form spelling confirmation |
| SRC-010 | PubChem (NIH) | Chemical properties · CAS numbers |
| SRC-020 | OSHA | Safety regulatory context |
| SRC-021 | ECHA | European chemical safety |

**Permanently blocked:**
- Market statistics without primary government or institutional source
- Safety thresholds as prescriptive advice
- Medical or therapeutic claims
- Chemical handling instructions of any kind
